Qick Summary

  • A simple website for users to take notes with local storage in browser. Therefore the notes will not disppear as browser fresh.
  • It's the project that was applied in single page and finished with vue-router.
  • I tried to design the brief layout and bring a good UX for users.
  • There are some basic but useful features in my design. Users can decide there customized tags and search notes with keywords easily.
  • A real backend database such as Firebase I want to intergrate to my note project with user authentication in the future.

Features

Notes List Main Page

Notes can be easily search by keyword or tags

  • All edited notes were saved in the browser's local storage until the note list page was loaded to get data in local storage.

Viewing Note

Viewing each note, editing and deleting is all available.

  • Using the unique id (uuid library), which was installed by npm to build the path in vue-router rendering.

Editing note

Editing each note

  • Editing page was finished easily by two-data binding feature in Vue.js.
  • I have used "set" instead of "array" in JS to achieve the unique value when users added new repeated tags.

Mobile Application

RWD design

  • 80% pure CSS to achieve RWD design.
  • All features on the large screen are also available on the mobile device.

  • Currently, the website is available with GitPage and analzed with Google analysis.

Features

Viewing Note

View the note in the single page, edit and delete is also available.

  • Vue-cli
  • Packages installed with npm
  • Single page application with vue-router

Editing note

Edit the note through data binding feature of Vue.js framework.

  • Vue.js / v-model

Classification with Tags

Notes can be quickly search by uses' tags.

  • Vue.js / v-model

Keywords search

Keywords can be easily search with title, tag or text in any notes.

  • Vue.js / v-model

Mobile Application

The website is RWD in mobile device.

  • CSS / SASS
  • RWD

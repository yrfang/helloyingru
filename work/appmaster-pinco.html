Qick Summary

  • A website platform, which provides services for users to find their interested sport courses, coaches or places.
  • Using React.js and React-router to build the website view.
  • Using Redux with immutable format to manage states and props between each container.
  • Familiar with backend API communication and connection.
  • Experienced in Google map, Facebook login, Third-Party Payment and etc.

Features

Home Page

Homepage with about us section

  • Users can see some new course information in homepage.
  • Users can also choose their interested course to see the detailed info about the course.

Course Category Page

Viewing different customized searching condition for courses

  • Using the React Semantic UI to implement time and category dropdown menu.
  • Searching courses by time, course category or keywords are all available.

Course Detailed Page

Viewing cuorse detailed information with booked state

  • Viewing course information such as course's introduction, location and coach's info...etc.
  • A booked button showing user's booked state.
  • If course's booked state is available, user can book can get payment page with redirect to Spgateway service.

Member Login

Email or Facebook login

  • Pinco website has its own member management system, including third-party social login such as Facebook.
  • Pinco members can login to view user's booked recording lists.

Member Info Page

Member info can be saved after modifying

  • Pinco member can modified some user's information, such as cover photo, birthday, password...etc.

Member Records Page

Member can view his/her booked recording lists

  • Viewing member's booked transation information.
  • There are five transaction type of tabs can be filtered in recording page.
  • All transaction list can be furthermore viewed in transaction page.

Desktop/Mobile Application

RWD

  • All features are available on desktop, pad, or mobile devices.
